04 - EFI Calibration

  • Show All Code
  • Hide All Code

For a time of interest from THEMIS or MMS (or RBSP or other), based on the overview plots at the mission website, finish the following steps.

  • Obtain EFI data in spinning and despun spacecraft coordâ€™s and in some geophysical (e.g., GSE or GSM) coordâ€™s.
  • Using the data in DSL or DMPA coordinates (where XY are despun, spin-plane components) estimate the offset in X and Y from a quiet time in the magnetosphere when the flows, and hence â€“VxB, are expected to be slow. Remove the offset from the data.
  • With the spin plane offsets removed, obtain the 3rd component (Z) for times when the Bfield is at a large angle relative to the spin plane (>10 deg or more) using the E*B=0 approximation.
  • Compare your estimate of Ez with the measurement of the same component made by the axial electric field boom pair. How large are the offsets for the axial field compared to those you have found for the spin plane components?

Note: A crib sheet and a jupyter notebook on how to perform the offset removal and EdotB determination are my_thm_efi_load_cal_crib.pyspd and my_thm_efi_load_cal_crib.py, located in the class notes below Lecture 12. def calibrate_offset(data, offset):
    return data - offset
Code 
def calibrate_e_dot_b(e_data: xr.DataArray, mag_data, threshold_angle=10):
    data = e_data.copy()
    data[:, 2] = (
        -(e_data[:, 0] * mag_data[:, 0] + e_data[:, 1] * mag_data[:, 1])
        / mag_data[:, 2]
    )
    
    threshold = np.sin(np.deg2rad(threshold_angle))
    mask = np.abs((mag_data[:, 2] / mag_data[:, 3])) > threshold
    data[~mask] = np.nan
    
    return data
