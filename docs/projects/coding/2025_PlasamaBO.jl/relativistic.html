Relativistic Plasma Dispersion Function

The relativistic parallel integral is no longer a “one-Maxwellian–(Z())” object. The key reason is that (i) the independent variable should be momentum (p_) (not velocity), and (ii) the resonance denominator acquires ()-dependence through (v_(p_)) (and in the magnetized case also (s/)). That breaks the linear-in-(v) structure that makes the nonrelativistic (Z()) so convenient.
